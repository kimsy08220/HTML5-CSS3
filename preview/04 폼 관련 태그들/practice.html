autofocus : 마우스 커서 표시
placeholder : 입력란에 표시하는 힌트
readonly : 입력하지 못하게 함
required : 필수 필드 체크
min, max, step : 해당 필드의 최솟값, 최댓값, 숫자 간격
(단, type이 date, datetime, datetime-local, month, week, time, range, number만 가능)
size, minlength, maxlength : 몇 글자까지 보이게 할지 결정, 입력 가능한 최소 글자, 최대 글자
select(multiple), option(selected) : 드랍다운 목록 표시(중복 선택 가능), 여러 옵션 표시(기본으로 선택되어 있는 옵션 지정)
optgroup : 여러 항목 그룹을 묶을 때 사용(label 속성을 사용해 그룹 제목을 붙임)
textarea, cols, rows : 여러 줄의 텍스트 입력, 가로 길이, 세로 길이 지정
button : 다양한 형태의 버튼 삽입(image도 가능)

신청자

output : 계산 결과를 브라우저에 표시

+ =

progress : 작업 진행 상태를 브라우저에 표시 meter : 전체 크기 중에서 얼마나 차지하는지를 표현할 때 사용(예. 하드디스크 사용량, 유권자 투표율)
